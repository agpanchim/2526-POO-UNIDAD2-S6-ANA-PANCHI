# Nombre del Programa: Servicio de Taxi 

## Descripción
Este proyecto implementa un sistema básico de servicio de taxi, aplicando los principios fundamentales.
El sistema permite solicitar una unidad de taxi y asignar el taxi disponible más cercano a la ubicación del usuario.

## Objetivo
Aplicar los conceptos de Programación Orientada a Objetos, tales como:
- Clases y objetos
- Herencia
- Encapsulación
- Polimorfismo
- Organización del proyecto en módulos y paquetes

## Conceptos de POO Aplicados

### Herencia
Se utiliza una clase base `Vehiculo` y una clase derivada `Taxi`, donde `Taxi` hereda atributos y métodos de `Vehiculo`.

### Encapsulación
El estado de disponibilidad del taxi se encuentra encapsulado mediante un atributo privado, accedido únicamente a través de métodos.

### Polimorfismo
El método `obtener_estado()` es sobrescrito en la clase `Taxi`, permitiendo comportamientos diferentes según el tipo de objeto.
